Detailed Description

The Likelihood aggregators maintains the current estimate of the log-likelihood of the current token assignments.

llik_topics = ... ndocs * (gammaln(ntopics * alpha) - ntopics * gammaln(alpha)) + ... sum_d(sum_t(gammaln(n_td + alpha)) - gammaln(sum_t(n_td) + ntopics * alpha));
