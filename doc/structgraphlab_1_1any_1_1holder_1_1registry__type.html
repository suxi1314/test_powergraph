template<typename ValueType>
struct graphlab::any::holder< ValueType >::registry_type

The following struct defines the static member used to automatically register the deserialization function for this holder type and cache a shared id used to quickly identify the deserialization function.

Note that the registry actually uses the NAME of the type so renaming a type will result in an incompatible deserialization.
