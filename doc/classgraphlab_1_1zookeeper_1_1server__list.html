Detailed Description

A simple zookeeper service to maintain a list of servers. The service provides the ability to watch for new servers leaving and joining the service through the use of callbacks.

void graphlab::zookeeper::server_list::set_callback ( boost::function< void(server_list *cur, std::string name_space, std::vector< std::string > server) >  fn)

Adds a callback which will be triggered when any namespace in the prefix changes. The callback arguments will be the server_list object, the namespace which changed, and the new list of servers in the name space. Calling this function will a NULL argument deletes the callback. Note that the callback may be triggered in a different thread.
